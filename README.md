
# 💔 Apology Website

An interactive, emotional apology experience with anime-style character animations and interactive slapping mechanics.

## Features

✨ **Anime Character Animation** - A fully animated character drawn with canvas
🎭 **Dynamic Expressions** - Character emotions change based on damage level
💔 **Interactive Slapping** - Click to slap, character shows damage and emotions
📝 **Three-Part Speech** - Navigate through heartfelt apology speeches
🎨 **Beautiful UI** - Dark theme with glassmorphism effects
🎵 **Audio Support** - Background music and sound effects
💗 **Victory Animation** - Celebration with heart particles
📱 **Responsive Design** - Works on mobile and desktop

## How to Use

### Setup

1. **Clone or download this repository**
2. **Add audio files** (optional):
   - `audio/background-music.mp3` - Ambient background (optional)
   - `audio/slap-sound.mp3` - Slap sound effect (optional)
   - `audio/forgive-sound.mp3` - Forgive success sound (optional)
3. **Open `index.html` in a web browser**

### How It Works

1. **Read the Speech** - Navigate through 3 parts of the apology using Previous/Next buttons
2. **Express Your Anger** - Slap the character as many times as you want
3. **See the Damage** - Character shows emotional and physical damage
4. **Unlock Forgiveness** - After enough slaps, the "Forgive Me" button becomes available
5. **Celebrate** - Click forgive and watch the victory animation

### Controls

- **SLAP HIM** button or **click the character** - Slap the character
- **← Previous / Next →** - Navigate speech parts (or use arrow keys)
- **FORGIVE ME** button - Accept the apology (or press Enter)
- **Spacebar** - Slap quickly
- **Arrow Keys** - Navigate speech
